# 示例模块

## 概述

示例模块提供了一个基础的示例接口，用于演示 API 的基本功能。

## API 列表

### 示例首页

**请求信息：**

- 请求路径：`/api/v1/demo/`
- 请求方法：`GET`
- 请求头：
  ```http
  Authorization: Bearer your-token
  ```

**响应结果：**

```json
{
    "code": 200,
    "message": "success",
    "data": {
        "title": "示例模块",
        "description": "这是一个示例接口",
        "version": "1.0.0"
    }
}
```

## 错误码说明

| 错误码 | 说明 |
|--------|------|
| 400 | 请求参数错误 |
| 401 | 未认证或认证失败 |
| 403 | 无权限访问 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |

## 注意事项

1. 这是一个示例模块，用于演示 API 的基本功能
2. 实际项目中可以根据需要扩展更多功能
