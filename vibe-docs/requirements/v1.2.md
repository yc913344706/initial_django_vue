# V1.2 版本需求文档

## 概述

@code/frontend/src/views/system/user/detail.vue 这是用户详情页的前端。
@code/backend/apps/user/views.py 这是用户详情页的后端。

我看现在的功能，是不支持用户修改密码的，所以我想增加：
1. 用户自己可以修改密码
2. 管理员可以重置密码

另外，用户配置增加：配置密码的复杂度要求，可以在下面这两个文件中增加：
- 后端： @code/backend/apps/user/views.py
- 前端： @code/frontend/src/views/system/user/config.vue
